Manual de Gestion del Panel

Version facil, para uso diario del equipo.

Objetivo

Este manual explica como usar el panel para atender chats, asignarlos, responder manualmente y registrar pedidos procesados.

1. Ingreso al panel

  1. Abrir el navegador.
  2. Entrar a http://localhost:3000.
  3. Ingresar usuario y contrasena de admin.

Si no inicia sesion, revisar usuario/hash en .env.

2. Flujo recomendado por chat

  1. Seleccionar una conversacion en la barra izquierda.
  2. Presionar Tomar chat.
  3. Si necesitas escribir vos, presionar Pausar bot.
  4. Enviar mensaje manual.
  5. Al terminar, presionar Marcar procesado.
  6. Opcional: Liberar para que otro admin lo tome.

3. Reglas de operacion

  • Un chat debe estar asignado para poder operarlo.
  • Si el chat esta asignado a otro admin, se puede ver pero no operar.
  • El conteo de reportes sube al marcar Procesado.
  • Evitar responder manualmente sin pausar si no hace falta.

4. Reportes

En la barra superior se muestran tres metricas:

MetricaSignificado
Procesados diaChats marcados procesados hoy.
Procesados semanaChats marcados procesados en la semana actual.
Procesados mesChats marcados procesados en el mes actual.

5. Problemas comunes

No puedo enviar mensaje

  • Verificar que el chat este asignado a tu usuario.
  • Verificar que esta pausado si vas a responder manual.

No puedo tomar chat

  • Puede estar asignado a otro admin.

No se actualizan metricas

  • Solo cambian cuando se marca el chat como procesado.

6. Como imprimir en PDF

  1. Abrir este archivo HTML en el navegador.
  2. Presionar Ctrl + P.
  3. Elegir Guardar como PDF.
  4. Guardar el archivo.
